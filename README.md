# Digital Isai - Autonomous WhatsApp Music Backend

**100% WhatsApp inside music production system** for Tamil songs.

## Quick Start (Termux)

```bash
pkg install python ffmpeg ngrok -y
pip install -r requirements.txt
cp .env.example .env
ngrok http 8000
python main.py
```

## Features
- AI Sales Closer (Ananya)
- 30 min timeout + Parallel Pitching
- Auto 45-sec preview
- Razorpay splits + Escrow
- Auto PDF Invoice
